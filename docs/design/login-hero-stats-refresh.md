# 登录页左侧数据与描述更新

## 背景

登录页左侧品牌区统计与描述为硬编码，热点数量与渠道表述已落后于库内实际数据。

## 方案

对照 MySQL 现状（2026-07-14）：

| 指标 | 库内 | 登录页展示 |
|---|---|---|
| 一级品类 | 5 | 5 |
| 热点链接 | 724 | 700+ |
| 研究来源 (`source_materials`) | 84 | 84 |

描述由「Amazon、YouTube、Reddit、Google Trends」改为「Amazon、小红书、YouTube、Reddit 等渠道」，与当前 `hot_links` 平台分布一致。

## 使用说明

前端静态文案；数据量显著变化时需再次人工同步，或后续改为公开只读统计接口动态拉取。
