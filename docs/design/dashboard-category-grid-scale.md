# 总览品类卡片墙可扩展布局

## 背景

总览页品类区原先用 6 列 Bento（前 2 卡 span 3、后 3 卡 span 2），只适配 5 个一级品类。扩到 20–30 个时：

- 第 6 张起无跨列规则，会变成 1/6 宽窄条
- 全量大卡会把视频/讨论/关键词顶出首屏

## 方案

均匀三列网格 + 活跃度排序 + Top 12 截断：

| 品类数 | 表现 |
|---|---|
| 5（当前） | 固定 3 列 → 自然 3+2，卡片等宽，无「查看全部」 |
| ≤12 | 全部展示，多行三列 |
| >12 | 只展示活跃度 Top 12，「查看全部」跳转品类百科 |

活跃度分：`hotLinks * 2 + trends + sources`。

## 核心改动

- `frontend/src/views/DashboardView.vue`：`visibleCategoryRows` / `hasMoreCategories`、`browse` 事件
- `frontend/src/styles.css`：去掉 `nth-child` 跨列，改为 `repeat(3, 1fr)` + 响应式 2/1 列
- `frontend/src/App.vue`：`@browse` → `activeView = 'encyclopedia'`

## 使用说明

超过 12 个一级品类时，总览自动截断；完整目录仍走左侧品类树或「品类百科」。

## 子品类展示（2026-07-13）

有子品类的一级品类（当前：热疗 → 肩颈热敷、远红外热疗）在总览中：

1. 父卡仍展示聚合后的热点/趋势/来源
2. 子卡紧跟在父卡之后，虚线边框 + 顶部父品类名标注
3. 排序规则：一级按活跃度，其子类挂在对应父卡后面
