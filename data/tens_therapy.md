## 一、品类基础信息

### 1.1 品类名称与定义

**品类名称**: TENS 电疗（经皮神经电刺激）  
**英文名称**: Transcutaneous Electrical Nerve Stimulation (TENS / TNS)  

**定义**: TENS 是一种通过皮肤表面电极输送微弱电流以刺激神经从而达到治疗目的的设备。TENS 通过调节脉冲宽度、频率和强度来缓解急性和慢性疼痛。设备通常使用导电凝胶垫将两个或更多电极连接到皮肤上。典型的电池供电 TENS 设备能够调节脉冲宽度、频率和强度。

> 来源: [Wikipedia - TENS](https://en.wikipedia.org/wiki/Transcutaneous_electrical_nerve_stimulation)

### 1.2 产品类型

| 类型 | 描述 | 典型频率 | 适用场景 |
|------|------|----------|----------|
| **传统型 TENS（高频）** | 高频刺激（>50 Hz），强度低于肌肉收缩阈值，产生感觉强度刺激 | 80-150 Hz | 慢性疼痛日常管理 |
| **低频 TENS（针刺型）** | 低频刺激（<10 Hz），强度足以引起肌肉收缩 | 1-10 Hz | 深层组织疼痛、运动损伤 |
| **混合频率 TENS** | 交替使用高低频，减少重复使用导致的耐受性 | 混合 | 长期疼痛管理 |
| **便携式 TENS** | 小型电池供电，可穿戴，适合外出使用 | 可调 | 工作或旅行时使用 |
| **临床型 TENS** | 更大功率，多通道输出，专业调控 | 可调 | 物理治疗诊所 |

> 来源: [Wikipedia - TENS](https://en.wikipedia.org/wiki/Transcutaneous_electrical_nerve_stimulation); [rehatora.net](https://rehatora.net)

### 1.3 产品边界

**包含**:
- 便携式 TENS 设备（电池供电）
- TENS 电极贴片（导电凝胶垫）
- TENS 专用导线/线缆
- 可穿戴 TENS 设备（如 TENS 皮带式、贴片式）
- 混合型 TENS+EMS 设备（以 TENS 为主功能）

**排除**:
- ❌ EMS（电肌肉刺激）设备——以肌肉锻炼/康复为目的，非神经疼痛缓解
- ❌ 微电流治疗（Microcurrent）设备——使用极低电流（μA级别），机制不同
- ❌ IFC（干扰电流疗法）设备——使用两路中频电流交叉产生低频效果
- ❌ 植入式神经刺激器（如脊髓刺激器 SCS）——需手术植入
- ❌ 普通按摩仪/按摩枪——使用机械振动而非电刺激

> 来源: [Wikipedia - Electrotherapy](https://en.wikipedia.org/wiki/Electrotherapy); [Wikipedia - Electrical muscle stimulation](https://en.wikipedia.org/wiki/Electrical_muscle_stimulation)

## 二、用户画像与场景

### 2.1 用户画像

**核心用户群体**:

1. **慢性疼痛管理者**
   - 特征: 患有骨关节炎、纤维肌痛、慢性腰背疼痛，长期依赖止痛药
   - 行为: 寻求非药物疼痛管理替代方案，关注副作用
   - 来源: 约44%的用户在使用TENS后报告疼痛显著减轻（Wikipedia引用临床研究）

2. **糖尿病神经病变患者**
   - 特征: 糖尿病导致周围神经病变，手足疼痛/麻木
   - 行为: 遵医嘱尝试非药物镇痛方案

3. **术后恢复期患者**
   - 特征: 手术后（如膝关节置换、剖宫产）需要缓解切口周围疼痛
   - 行为: 在康复期间短期使用，减少阿片类药物依赖

4. **运动损伤康复者**
   - 特征: 肌肉拉伤、扭伤、运动后肌肉酸痛
   - 行为: 短期使用加速恢复，与RICE方案配合

5. **月经疼痛女性**
   - 特征: 原发性痛经寻求非口服药物缓解方案
   - 行为: 月经期间短期使用

> 来源: [Wikipedia - TENS Medical Uses](https://en.wikipedia.org/wiki/Transcutaneous_electrical_nerve_stimulation#Medical_uses); [Harvard Health](https://www.health.harvard.edu/pain/transcutaneous-electrical-nerve-stimulation-tens-for-pain-management)

### 2.2 品类使用场景

**核心场景**:

| 场景 | 描述 | 使用频率 |
|------|------|----------|
| **居家疼痛管理** | 在沙发上或床上使用TENS设备缓解腰背/颈肩疼痛，每次20-30分钟 | 每日1-3次 |
| **工作间隙疼痛缓解** | 办公桌前使用便携TENS贴片缓解久坐导致的腰痛或颈痛 | 按需使用 |
| **运动恢复** | 运动后在大腿/小腿/肩部使用TENS加速肌肉恢复 | 运动后使用 |
| **夜间疼痛管理** | 睡前使用TENS缓解疼痛以助入睡 | 睡前使用 |
| **旅行途中** | 使用小型便携TENS设备在飞机/火车上缓解关节僵硬 | 旅行时使用 |
| **物理治疗辅助** | 在物理治疗诊所由治疗师使用临床型TENS设备 | 每周2-3次 |

> 来源: [Johns Hopkins Medicine](https://www.hopkinsmedicine.org/health/treatment-tests-and-therapies/transcutaneous-electrical-nerve-stimulation)

## 三、用户需求与品类痛点

### 3.1 用户基本需求

**功能需求**:
- 有效缓解急性和慢性疼痛（腰背痛、关节痛、神经痛等）
- 可调节刺激参数（频率、强度、脉冲宽度）以适应不同疼痛类型
- 便携、可独立操作
- 多通道输出可同时治疗多个疼痛部位

**场景需求**:
- 居家使用安全便捷
- 办公场景可隐蔽使用（贴片式可穿戴设备）
- 旅行时可携带

**社会需求**:
- 减少对止痛药物的依赖，避免药物副作用和成瘾风险
- 作为"主动自我管理"的健康行为体现

**潜在需求**:
- 与智能手机App连接，记录使用数据和疼痛评分
- 智能推荐最佳刺激参数
- 与其他健康管理数据整合

### 3.3 用户使用旅程

```
拆包 → 安装 → 试用 → 长时间使用 → 用后维护与管理
```

| 阶段 | 行为 | 痛点/需求点 |
|------|------|-------------|
| **拆包** | 打开包装，查看设备主机、电极贴片、导线、说明书 | 缺少清晰的快速入门指南；配件不全 |
| **安装** | 安装电池或充电，连接导线和电极贴片 | 导线接口标记不清晰；贴片首次粘贴位置不确定 |
| **试用** | 选择模式和强度，贴在疼痛部位，体验刺激感 | 初次使用强度过高引起不适；不知道应该贴在哪里 |
| **长时间使用** | 每天20-30分钟，定期更换贴片位置 | **贴片粘性随使用次数下降（核心痛点）**；电池续航不足；对特定频率产生耐受性 |
| **用后维护** | 清洁设备、更换贴片、收纳导线 | 替换贴片成本高（$10-20/包）；导线缠绕损坏；设备清洁不便 |

### 3.4 品类痛点、常见问题与困扰

**高频用户搜索问题**（反映核心痛点）:

1. **"TENS pad not sticking / TENS electrode pad adhesive loss"**
   - 🔴 贴片粘性下降是#1用户痛点
   - 需求: 更持久的粘合剂或可替换贴片的低成本方案
   - 来源: Amazon评论高频主题

2. **"How to extend TENS pad life / TENS pad lifespan"**
   - 🔴 贴片寿命短（通常10-20次使用后粘性大幅下降）
   - 需求: 更耐用的电极贴片材料

3. **"TENS unit not working / no sensation"**
   - 导线断裂、电池电量不足、贴片导电凝胶干涸
   - 需求: 设备故障诊断提示功能

4. **"TENS machine side effects / TENS safety"**
   - 皮肤刺激、过敏反应、肌肉不适
   - 需求: 低过敏性贴片材料

5. **"Where to place TENS pads"**
   - 用户不确定电极放置位置
   - 需求: 附带电极位置图指南或App指引

6. **"TENS vs EMS difference"**
   - 用户混淆TENS和EMS的功能差异
   - 需求: 清晰的品类教育和产品定位

> 来源: Amazon高频评论主题 + 用户搜索行为分析

## 四、技术与设计原则

### 4.1 品类核心技术与作用机制

**核心技术: 经皮神经电刺激**

**作用机制（双重镇痛机制）**:

1. **门控理论（Gate Control Theory）**
   - TENS刺激大直径Aβ感觉神经纤维
   - 这些纤维的信号"关闭"脊髓背角的"疼痛门"
   - 阻止小直径Aδ和C纤维的疼痛信号向大脑传递
   - 高频TENS（>50Hz）主要依赖此机制
   > 来源: [Wikipedia - Gate Control Theory](https://en.wikipedia.org/wiki/Gate_control_theory)

2. **内啡肽释放机制**
   - 低频TENS（<10Hz）刺激触发中枢神经系统释放内啡肽
   - 内啡肽是天然镇痛物质，效果类似阿片但无成瘾性
   - 作用较慢但持续时间更长
   > 来源: [Wikipedia - TENS](https://en.wikipedia.org/wiki/Transcutaneous_electrical_nerve_stimulation)

3. **神经调控机制**
   - TENS降低脊髓背角神经元的敏化
   - 提升GABA（γ-氨基丁酸）和甘氨酸水平
   - 抑制胶质细胞激活
   > 来源: Wikipedia临床研究引用

**关键参数**:

| 参数 | 范围 | 说明 |
|------|------|------|
| 频率 | 1-150 Hz | 高频(>50Hz)用门控机制；低频(<10Hz)用内啡肽机制 |
| 脉冲宽度 | 50-200 μs | 影响刺激深度和感觉强度 |
| 强度 | 感觉强度至运动强度 | "强烈但舒适"是最佳镇痛强度 |
| 通道 | 1-4通道 | 双通道最常见，可同时治疗两个部位 |

### 4.2 材料科学

**电极贴片材料**:
- **导电凝胶**: 通常为水基或硅胶基导电凝胶，提供皮肤-电极界面导电
- **电极基材**: 碳膜或金属箔（银/银氯化物），均匀分布电流
- **粘合层**: 医用级水凝胶粘合剂，亲肤且可反复粘贴
- **背衬**: 无纺布或PU膜，绝缘且防水

**设备外壳材料**:
- ABS塑料（主流）：耐用、绝缘、成本可控
- 硅胶涂层（高端）：改善手感和防滑

**导线材料**:
- PVC或TPE绝缘铜线
- 耐弯折设计减少断裂

> 来源: [Wikipedia - TENS Device Types](https://en.wikipedia.org/wiki/Transcutaneous_electrical_nerve_stimulation#Device_types)

## 五、新兴趋势

### 5.1 技术发展趋势

1. **智能TENS设备**
   - 蓝牙连接手机App
   - App调节参数、记录使用数据
   - 疼痛评分追踪和个性化方案推荐
   - 代表品牌: TENS 7000智能版、PowerDot

2. **可穿戴TENS**
   - 贴片式一体化设计（无导线）
   - 超薄柔性电路板
   - 手机控制，隐形佩戴
   - 代表产品: PowerDot 2 Uno, TENS 7000可穿戴版

3. **混合模式TENS+EMS**
   - 一机两用（疼痛缓解+肌肉锻炼）
   - 自动切换TENS/EMS模式
   - 代表产品: Compex系列、Marc Pro

4. **无线TENS**
   - 无导线设计，蓝牙连接控制器
   - 贴片直接内置微型刺激器
   - 代表产品: Wireless TENS by TENS 7000

### 5.2 消费者偏好趋势

- **便携性**: 从大型台式设备向口袋大小便携设备转变
- **美观性**: 从医疗外观向消费电子产品外观转变
- **易用性**: 一键操作、预设模式、语音引导
- **无药疼痛管理**: 随阿片危机加剧，消费者更偏好非药物疼痛方案
- **自管理健康**: 消费者更愿意在家自我管理慢性疼痛

### 5.3 新兴子品类

- **TENS+热疗组合设备**: 同时提供电刺激和热敷
- **TENS腰带式**: 专为腰痛设计的集成穿戴式设备
- **TENS足底按摩器**: 专为糖尿病足部神经病变设计
- **TENS+按摩组合**: 电刺激配合机械振动
- **处方数字TENS**: FDA批准的处方App+设备组合
